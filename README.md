# &spades; Tecnologias 

Foi usado nesse projeto as tecnologias abaixo, junto com o Visual Studio Code e o Git:

- Node.js
- React
- React Native
- Expo 
- MongoDB
- Yarn
- Insomnia

# &clubs; Projeto

 O DevRadar é um projeto que visa conectar desenvolvedores próximos a você que trabalham com as mesmas tecnologias. -
